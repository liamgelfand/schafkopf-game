# Schafkopf Mobile App

React Native mobile application for Schafkopf game.

## Setup

1. Install dependencies:
```bash
npm install
```

2. For iOS:
```bash
cd ios && pod install && cd ..
npm run ios
```

3. For Android:
```bash
npm run android
```

## Project Structure

The mobile app shares game logic types with the web version. Core game logic is handled by the backend API.
